# Excelsior Project (Luxor)

Welcome to the **Luxor** repository. This is the main repository holding the infrastructure, smart contracts, and interfaces for the Luxor modular sovereignty protocol on the Solana blockchain.

## Directories

- **`/admin-dashboard`**: The main management dashboard for viewing network analytics, balances, and operational metrics.
- **`/landing-page`**: The public-facing website highlighting the Luxor ecosystem, utility token (LXR), and project vision.
- **`/smart-contracts`**: The core on-chain programs and Solana smart contracts that power the modules.
- **`/pos-integrations`**: Point-of-Sale (POS) and real-world merchant integration tools.
