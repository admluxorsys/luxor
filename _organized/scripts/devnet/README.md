# 🔍 Devnet Scripts - Auditoría y Testing

Estos scripts están configurados para **Devnet** (hardcodeado).

## Scripts de Auditoría
- `audit_unique.js` — Audita wallets únicas, valida balances LXR/XLS
- `audit_master.js` — Auditoría completa (master version)
- `audit_aggressive.js` — Auditoría agresiva con reintentos
- `audit_backup.js` — Backup de auditoría

## Scripts de Escaneo
- `deep_scan.ts` — Escaneo profundo de datos
- `deep_scan_efficient.js` — Versión optimizada
- `robust_scan.ts` — Escaneo robusto con manejo de errores
- `final_scan.js` — Escaneo final

## Scripts Especializados
- `final_controlled_burn.js` — Quema controlada de tokens
- `find_largest.js` — Encuentra wallets con mayores balances
- `full_audit.js` — Auditoría exhaustiva
- `test.js` — Test básico
